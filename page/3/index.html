仅仅通过繁衍后代，我们的自我真的有一部分延续下去了吗

虚舟   2019年7月7日, 星期日

人类繁衍后代的最直接原因是性本能，但人们并不止于这种解释，他们还为自己的行为寻找更高大上的理由和意义，比如，为了传宗接代，为了种族的延续等等，但我觉得为了种族之类的比较扯，人是自私的动物，更多的是为了自己，他们把自己的后代当作自己的一部分，在自己死后这一部分继续存活下去，虽然看不到死后的情景，但一想到自己的子子孙孙一代代地延续下去，内心就获得了莫大的安慰和满足，我把这种意识称为自我延续的意识。

人类试图通过两种方式进行自我延续，一种是通过繁衍后代，另一种是是通过著书立说（广义地说，自从人类进入信息时代以来，任何生产制造信息的行为都应包括在内，为了行文方便，这里以“著书立说”代之）。前一种是物质上的延续，后一种是精神上的延续。

通过繁衍后代，你的基因会传递下去；通过著书立说，你的灵魂、意识、思想和人格的一部分会被他人吸收融合到其精神意识之中，从而得以延续。

通过繁衍后代，自我延续的对象只能是自己的后代，数量有限，受众面极窄；通过著书立说，自我延续的对象可以是任何人，数量巨大，受众面广阔。

有人囤积远远超出自己所需的财富，皆是为了其子孙后代，而其后代可以无穷无尽，所以其囤积财富的欲望也就永无尽头，因此贫富极化就产生了，这就是把繁衍后代当作自我延续的观念的产物。假如有人没有留后的观念，以著书立说作为人生的目的和自我延续的途径，因为每一个人都是TA潜在的读者，是TA自我延续的媒介和载体，所以TA会爱每一个人，而不是仅限于自己的后代，这就是因自私而博爱的情形。

我经常在得出结论之后，又对其产生怀疑。

有人说繁衍后代的目的是为了自我延续，因此问题来了，何为自我？由上天赋予无法选择不得不接受的躯壳和可以由自己自由塑造的灵魂，哪一个更能代表自我？通过繁衍后代而传递下去的基因真的能代表我们自己吗？我们的后代也许在生物性状上与我们有相似的地方，但是其灵魂、意识和人格却是独立的，无法通过基因传递和继承。

因此，仅仅通过繁衍后代，我们的自我真的有一部分延续下去了吗？

时间的遗民

虚舟   2019年7月5日, 星期五

有一天上帝喝醉了酒，

无意中训斥了时间一顿，

时间一生气就跳过了2019年六月的某一天，

很多人因此被遗弃在过去，

他们在往昔那笼罩着迷雾的森林和城市中苦苦探寻、徘徊，

却始终找不到通往未来的路，

他们是时间的遗民，

上帝酒醉的受害者。

暴风雨前夕的假面派对

虚舟   2019年6月28日, 星期五

收到同学通知，我去邻村中心广场参加一场假面派对，他说去的人都是认识的同学。我到的时候，那里已经有一些人了，广场上放着音乐，像莫扎特的风格，大家随着音乐的节奏起舞，跟我派对的是一个戴猫头面具，穿蓝黑色外衣，身材瘦小的人……不知什么时候，天色突然变暗了，豆大的雨点零零散散地落下来，忽然有人喊“龙卷风！龙卷风！”大家停止跳舞，纷纷摘下面具，只见远处的天空一片昏暗，一棵伸展到云层中的黑色巨树，正缓缓地向我所在的村子移动。这时那个跟我派对的人也摘下了面具，没想到是一个面容清秀的女孩子，可是她并不是我的同学，她可能发现了我正在看着她，侧过脸冲我微微一笑，瞬间，五颜六色的烟花在我心里升腾而起，照亮了那颗一直疲惫、焦虑又压抑的心，那一刻我忘记了一场暴风雨正要来临。

人生的目的与道路

虚舟   2019年6月27日, 星期四

理想的人生应该是先设定好目的，再依此规划相应的道路。然而实际情况往往是，人生的道路已经被规划好了，目的却还没有找到。因为在确定人生目的之前，我们需要先走一段路，实地勘察一下周围的环境和地貌，体验一下生活，有了人生经验然后才能做出决定。因此我们人生的目的往往是由走过的道路决定的，然而我们并没有多少可供选择的道路，这些道路大都已经被历史、社会、家庭和种种条件设置和限定好了，所以我们并没有多少自由，只能在已经被长辈和社会规划好的路上无奈地前行，很多人把道路延伸的方向当作自己人生的目的，而不是根据人生的目的确定道路的方向，这样的人将度过幸福顺遂的人生。如果你不幸有另外不同的目的，却没有勇气离开人群去寻找自己的方向，那你将会感到痛苦，最终带着遗憾走完一生。

在思考人生的目的的时候，我想起了两个人的话，一个是王小波说的，“我活在世上，无非想要明白些道理，遇见些有趣的事。倘能如我所愿，我的一生就算成功。”另一个是孔子说的，“朝闻道，夕死可矣。”其实他俩说的是同一件事，把明白些道理（闻道）作为自己的人生目的，只不过小波说得朴素又低调，孔子说得委婉却夸张。换一个高大上且人人熟知的说法就是，追求真理，这话从科学家嘴里说出来，我们觉得很正常，可是从平民百姓嘴里说出来，就有点疯疯癫癫的。所以作为一个普通人，我更喜欢小波的说法。

大学学习高数的时候，总觉得实数这东西云山雾罩，不弄清楚它的结构心里总不舒服，于是一头扎进去，搜集各种相关的理论，一层层地追根朔源，最后卡在了集合论的一个定理上，死活弄不懂了。这个世界广阔无垠，自己却陷身于一粒沙子之中而无法前进，深深地体会到一种无力感和自己的蠢笨。人生的路上总是散布着一团团的迷雾一个个的沼泽，人需要清醒地认识到自己的能力有限，你不可能驱散所有的迷雾，排干每一个沼泽，他应该记住自己的目的是什么，而不是疯了似地跟每一团迷雾每一个沼泽过不去。

没有目的人生，浑浑噩噩，性价比很低，然而目的性太强的人生，犹如只顾赶路的旅途，会错过路上的美景。

解决 Windows Thin PC 系统中 Firefox 和 IE 播放音频和视频的问题

虚舟   2019年4月9日, 星期二

今天为我的 Windows Thin PC 系统安装了一些功能组件，解决了一系列问题。

使用 Thin PC 一段时间了，有一个问题一直困扰着我，我的 Thin PC 上的 Firefox / IE11 无法播放一些 HTML5 音频和视频（但 Chrome 可以），比如，我博客上的 HTML5 音频标记为“错误：音频类型不受支持或文件路径无效”，Twitter 上的视频则提示“我们无法在这个浏览器上播放视频，请尝试换用其它浏览器。”然而 Youtube 上的视频则可以播放，奇怪的很。因为这个原因再加上 Firefox 在我这台 Thin PC 电脑上的速度不如 Chrome 快，所以一直在用 Chrome。然而最近发现 Firefox 升级后打开我博客的速度要比 Chrome 快（可是在另一台配置较高的 Win10 上则相反），因此打算在我的 Thin PC 电脑上主要使用 Firefox，于是经过一番查找终于找到了问题的症结所在： Windows Thin PC 没有安装相应的音频和视频组件。

经测试，安装如下组件可解决上述在 Firefox 中无法播放 Twitter 视频的问题，但 IE 中问题依旧。

  1. winemb-premiumcodecs-dolby-ac3-audioencoder.cab
  2. WinEmb-PremiumCodecs-MPEG2andDolbyDecoder.cab
  3. WinEmb-PremiumCodecs-MPEG2-Decoder.cab
  4. winemb-premiumcodecs-mpeg2-encoder.cab
  5. WinEmb-PremiumCodecs-MPEG3.cab
  6. WinEmb-PremiumCodecs-MPEG4.cab
  7. winemb-premiumcodecs-wmv.cab

安装 winemb-media-support.cab 组件解决了有些 HTML5 音频和视频在 Thin PC 的 Firefox / IE11 中无法播放的问题。

另外安装 winemb-photos-viewer.cab 组件可解决 Thin PC 不使用第三方图片浏览软件无法查看图片的问题，安装这个组件后电脑中图片的右键菜单会出现两个“预览”，我用右键管家解决了这个问题。

具体安装步骤和方法参照如下文章：

  • 开启 Windows Thin PC 被关闭的 Windows 功能
  • Windows Thin PC 安装功能组件
  • Windows Thin PC 中文化

再安装去广告和捆绑的 Adobe Flash Player，Thin PC 系统中 Firefox 应该就可以播放几乎所有网页的音视频了，这就完美了。
